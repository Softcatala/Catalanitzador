• El Gimp

    El GIMP és un programa lliure de retoc fotogràfic i edició d'imatges. Es pot utilitzar com a simple programa de dibuix, com a programa de retoc fotogràfic professional, com a sistema de processament massiu en línia, com a dissenyador massiu d'imatges de producció, com a convertidor de formats gràfics, etc. Està disponible per al Linux, Windows i Mac.

    Per obtenir més informació aneu a: http://www.softcatala.org/

  • LibreOffice

    El LibreOffice és un paquet ofimàtic lliure que inclou un processador de textos (Writer) compatible amb el Microsoft Word, un gestor de fulls de càlcul (Calc) compatible amb el Microsoft Excel, un editor de presentacions (Impress) compatible amb el Microsoft PowerPoint, un editor de bases de dades (Base) que pot importar bases de dades del Microsoft Access, un editor de dibuixos vectorials (Draw) i un editor de fórmules matemàtiques (Math).

    Per obtenir més informació aneu a: http://www.softcatala.org/

  • Inkscape

    L'Inkscape és una eina lliure de dibuix vectorial amb capacitats semblants a l'Illustrator, el Freehand i el CorelDraw. Permet crear formes bàsiques, text, marcadors, clons, composició de transparències, degradats i agrupacions, edició per nodes, capes, operacions complexes amb camins, entre moltes altres funcions.

    Per obtenir més informació aneu a: http://www.softcatala.org/

  • Ubuntu

    L'Ubuntu és una de les distribucions més populars de GNU/Linux, amb milions d'usuaris arreu del món. És un sistema operatiu complet, basat en l'escriptori GNOME i amb l'entorn Unity, completament traduït al català. Disposa de més de 25.000 programes gratuïts fàcilment instal·lables, els principals dels quals disponibles en català.

    Per obtenir més informació aneu a: http://www.ubuntu.cat/
