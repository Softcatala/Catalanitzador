• GIMP

    El GIMP és un programa lliure de retoc fotogràfic i edició d'imatges. Es pot utilitzar com a simple programa de dibuix, com a programa de retoc fotogràfic professional, com a convertidor de formats gràfics, etc. És una alternativa lliure a l'Adobe Photoshop. Està disponible per al Linux, Windows i Mac.

    Més informació: https://www.softcatala.org/

  • LibreOffice

    El LibreOffice és un paquet ofimàtic lliure que inclou un processador de textos (Writer), un gestor de fulls de càlcul (Calc), un editor de presentacions (Impress), un editor de bases de dades (Base), un editor de dibuixos vectorials (Draw) i un editor de fórmules matemàtiques (Math). És una alternativa lliure al Microsoft Office (Word, Excel, PowerPoint i Access) i és compatible amb els seus formats. Està disponible per al Linux, Windows i Mac.

    Més informació: https://www.softcatala.org/

  • Inkscape

    L'Inkscape és una eina lliure de dibuix vectorial amb capacitats semblants a l'Illustrator, el Freehand i el CorelDraw. Permet crear formes bàsiques, text, marcadors, clons, composició de transparències, degradats i agrupacions, edició per nodes, capes, operacions complexes amb camins, entre moltes altres funcions.

    Més informació: https://www.softcatala.org/

  • Ubuntu

    L'Ubuntu és una de les distribucions més populars de GNU/Linux, amb milions d'usuaris arreu del món. És un sistema operatiu complet, basat en l'escriptori GNOME, completament traduït al català. Disposa de més de 25.000 programes gratuïts fàcilment instal·lables, molts dels quals disponibles en català.

    Més informació: https://www.ubuntu.cat/
